Add project to VCS (Git)

January 11 2018

First steps: Download Git.exe from official website https://git-scm.com/downloads and
install it on computer in default directory disc C:/Program files/....

Notice: Version control configuring as for Webstorm as for other products of JetBrain.
Where? Go to the setting in the left panel and find Version Control and select it
then select Git bookmark on which is Path to Git and specify root file git.exe.

After all actions press button Test to check if everything OK! If it's true appears green notification.
